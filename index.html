IOCL News Portal

Developed a comprehensive, responsive, and interactive news portal for Indian Oil Corporation Limited (IOCL), Barauni Refinery during my vocational training internship. The application was designed to provide employees with a centralized platform to access the latest refinery announcements, official notifications, project updates, recruitment news, safety alerts, financial reports, and monthly news in a clean, organized, and user-friendly dashboard.

The portal was built with a strong focus on usability, accessibility, and responsive design, ensuring a seamless experience across desktop, tablet, and mobile devices. A modern dashboard layout was created to simplify navigation while allowing users to quickly discover important information through dynamic content rendering and intuitive interface components.

Implemented a wide range of interactive frontend features using HTML, CSS, and JavaScript, including real-time search functionality, category-based filtering, quick filter cards, today's highlights section, notification center with unread counters, search history using Local Storage, bookmark management, voice search integration, weather widget, loading animations, smooth page transitions, responsive navigation, and dark/light theme support. These features were designed to improve accessibility, increase user engagement, and provide a modern application-like browsing experience.

To enhance performance and maintainability, the application follows responsive design principles, optimized asset loading, reusable UI components, modular JavaScript logic, and clean CSS architecture. Special attention was given to smooth animations, hover interactions, cross-browser compatibility, accessibility (WCAG-friendly practices), and mobile-first development to ensure a polished and professional user experience.

The project also integrates REST API-based dynamic content loading, allowing news data to be fetched and displayed efficiently while supporting future scalability and easy content management. Throughout development, emphasis was placed on creating an intuitive interface that balances functionality with visual appeal, closely resembling the experience of a production-ready enterprise dashboard.

  • REST API Integration for Dynamic News

  • Category-Based News Filtering
  • Notification Center with Live Updates
